# Filmorate (Фильмотека)
REST приложение, с помощью которого пользователи могут добавлять фильмы и друг друга в друзья. Также оценивать фильмы в базе путем добавления лайков.

# Стек технологий
- Java
- Spring Boot
- Hibernate
- PostgreSQL
- JdbcTemplate
- Apache Maven
- Lombok
- Postman
- IntelliJ IDEA
- DBeaver
